Projets de Programmation L2 MIASHS POO

Il vous est demandé, pour ce projet, de choisir un site web de type "e-commerce" ou tout autre site qui montre une collections d'éléments (Items) ayant des caractéristiques communes. Choisissez une page présentant un grand nombre d'Items. Pour chaque Item, vous devrez extraire les informations du code source de la page (1) puis analyser les informations extaites de la collection d'Items (2).

pour vous aider à coder, utilisez ce site pour générer le squelette du projet.

  • Choissez un site, puis vérifiez que les données peuvent être extraites en remplissant le champ "adresse", validant que les données sont correctement extraites dans le nouvel onglet
  • Choisissez les propriétés des Items à extraire (prix, poids, couleur, en promotion, marque, description...)
  • choissiez les connaissances que vous voulez extraire de la collection Items extraite (moyenne des prix, marque la plus fréquente, couple de mot de plus fréquent dans la description...)
  • Téléchargez le projet, ouvrez-le avec VS code. Codez dans les classes ItemAnalyzer et ItemsScrapper et exécutez toutes opérations avec la classe Main
